# Proyecto de Apuestas Interactivas

## Descripción
Este proyecto es una plataforma interactiva de apuestas deportivas, donde los usuarios pueden registrarse, iniciar sesión, realizar apuestas en diferentes deportes (fútbol, baloncesto, tenis, etc.) y competir en rankings. El sistema también permite crear grupos y seguir a amigos para participar en apuestas conjuntas.

## Tecnologías utilizadas
- **HTML5**: Para la estructura básica de las páginas.
- **CSS3**: Para el diseño y estilo visual de las páginas.
- **JavaScript**: Para las interacciones y lógica dinámica de la página.
- **Boxicons**: Iconos para mejorar la interfaz de usuario.
- **Git**: Control de versiones del proyecto.

## Instrucciones de instalación y uso
Iniciar sesión y registrarse
El proyecto incluye botones para redirigir a las páginas de Iniciar sesión y Registrarse. Estos formularios no están conectados a una base de datos, pero proporcionan una estructura para implementar el sistema de autenticación.

Haz clic en Iniciar sesión o Registrarse desde la página de inicio (index.html) o desde cualquier otra página donde se muestren los botones.
Esto redirigirá a las páginas correspondientes, donde se pueden agregar funcionalidades adicionales más adelante.

En el menu de arriba tienes el apartado de enlaces donde cada uno te manda a su respectica pagina. El de apuestas , usuarios y el de rankings.

Si pulsas en el logo te redigirira a la pagina Principal. 

Por ultimo la zona del header con el usuario y dinero te llevara a la pagina del perfil personal.

El resto de botones y perfeccionar el responsive se realizaran en futuras versiones. 

### Prerrequisitos
- Tener instalado un navegador web moderno (Chrome, Firefox, etc.)
- Un editor de texto como Visual Studio Code para modificar los archivos si es necesario.
